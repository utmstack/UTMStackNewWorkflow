JSON Input

Utmstack can process custom logs that are not defined within system modules. Send the logs in JSON format to port 8080 (TLS)

POST https://{{instance}}:8080/v1/logs
* Utm-Connection-Key: {{mask}}
* Utm-Log-Source: your log source (mandatory header)
* Content-Type: application/json
Body (any valid object array)
