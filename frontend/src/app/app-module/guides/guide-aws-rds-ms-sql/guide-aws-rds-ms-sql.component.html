AWS RDS MS-SQL

For this integration to work properly, first you need to configure the IAM user for UTMStack.

  1. 1 Open the Amazon RDS console at https://console.aws.amazon.com/rds/

  2. 2 In the navigation pane, choose Databases and then choose the DB instance that you want to modify.

  3. 3 Choose Modify.

  4. 4 In the Log exports section, choose the logs that you want to start publishing to CloudWatch Logs. You can choose error, agent, or both.

  5. 5 Choose Continue and then choose Modify DB Instance on the summary page.

  6. 6 Click on the button shown below, to activate the UTMStack features related to this integration
