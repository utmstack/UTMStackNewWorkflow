Configuring IAM User

  1. 1 Go to IAM configuration panel and click on "Users".

  2. 2 Add a new user filling the name and marking the access type "Programmatic access". Then click on Next.

  3. 3 Click on Attach existing policies directly. Find CloudWatchReadOnlyAccess and mark it. Then click on next.

  4. 4 In the tags page click on next.

  5. 5 Create user and then download the csv file with the access and secret keys.

  6. 6 Fill the following inputs with the info obtained in previous steps.

Configuring AWS Cloudwatch

  1. 7 In the CloudTrail panel, select “Create trail”.

  2. 8 Fill in the "Trail name" field.

  3. 9 Mark "Select all S3 buckets in your account".

  4. 10 Fill in the "S3 bucket" field. The name of the bucket must be unique in S3. Then click on “Create”.

  5. 11 Click on the name of the trail to edit.

  6. 12 Configure CloudWatch Logs.

  7. 13 Fill the group name and continue.

  8. 14 Click on “Allow” to grant CloudTrail permissions.

  9. 15 Click on the button shown below, to activate the UTMStack features related to this integration
