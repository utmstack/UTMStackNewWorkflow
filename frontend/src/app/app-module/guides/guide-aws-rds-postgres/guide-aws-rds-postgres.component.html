AWS RDS PostgreSQL

For this integration to work properly, first you need to configure the IAM user for UTMStack.
To publish PostgreSQL logs to CloudWatch Logs using the console:

  1. 1 Open the Amazon RDS console at https://console.aws.amazon.com/rds/

  2. 2 In the navigation pane, choose Databases, and then choose the DB instance that you want to modify.

  3. 3 In the Log exports section, choose the logs you want to start publishing to CloudWatch Logs.
    The Log exports section is only available for PostgreSQL versions that support publishing to CloudWatch Logs.

  4. 4 Choose Continue, and then choose Modify DB Instance on the summary page.

  5. 5 Click on the button shown below, to activate the UTMStack features related to this integration
