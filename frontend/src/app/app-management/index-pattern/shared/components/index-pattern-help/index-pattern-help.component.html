Index patterns allow you to bucket disparate data sources together so their shared fields may be queried in UTMStack.

Single data source

Index a single data source named log-west-001 so you can build charts or query its contents fast.

Multiple data sources

Group all incoming data sources starting with log-west* so you can query against all your west coast server logs.

Custom groupings

Specifically group your archived, monthly, roll-up metrics of those logs into a separate index pattern so you can aggregate historical trends to compare.
